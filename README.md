## <img src="https://raw.githubusercontent.com/Tarikul-Islam-Anik/Animated-Fluent-Emojis/master/Emojis/Objects/Gear.png" width="25" /> Key Features

HCore unifies all essential Minecraft server management tools into a single, sleek dashboard:

* **<img src="https://img.icons8.com/material-rounded/24/58A6FF/lock.png" width="16" /> Advanced Authentication**
  Secure login via Minecraft player name verification and session management.
* **<img src="https://img.icons8.com/material-rounded/24/58A6FF/scales.png" width="16" /> Punishment Management**
  Comprehensive tracking of active and past penalties with a built-in appeal system.
* **<img src="https://img.icons8.com/material-rounded/24/58A6FF/ticket.png" width="16" /> Support System**
  Categorized support requests (Tickets) with multi-media file upload capabilities.
* **<img src="https://img.icons8.com/material-rounded/24/58A6FF/discord-new.png" width="16" /> Discord Integration**
  Seamless one-click Discord account linking using OAuth2 synchronization.
* **<img src="https://img.icons8.com/material-rounded/24/58A6FF/paint-palette.png" width="16" /> Modern UI/UX**
  High-fidelity dark theme with fluid transitions and fully responsive layouts.
